# WarstormSpecManager

Easy bot & raid group builder for Wrath of the Lich King (3.3.5)

WarstormSpecManager is a lightweight addon that makes it easy to build party and raid groups with bots on servers that support player bots.

Instead of typing bot commands over and over, you build your group with a simple UI and click Build.

---

## Features

- Build party or raid groups using class counts
- Premade raid setups for:
  - Naxxramas (10 / 25)
  - Ulduar (10 / 25)
  - Trial of the Crusader (10 / 25)
  - Icecrown Citadel (10 / 25)
- Save your own custom group setups
- Automatically adds bots one by one at a safe interval
- Automatically converts party to raid when needed
- Opens together with the main spec manager window
- No libraries, no dependencies

---

## Installation

1. Download or clone this repository
2. Extract the folder into:
   World of Warcraft/Interface/AddOns/
3. Make sure the folder name is:
   WarstormSpecManager
4. Start the game and enable the addon

---

## How to Use

1. In game, type:
   /wssm

2. Two windows will open:
   - Main window (specs, autogear, etc.)
   - Bot group builder window

3. Choose one of the following:
   - Select a raid preset from the dropdown
   - OR manually enter how many of each class you want

4. Click Build

Bots will start joining automatically.

---

## Saving Group Setups

- Enter a name in the Comp name box
- Click Save
- Your setup will be available next time you log in

---

## Requirements

- World of Warcraft Wrath of the Lich King 3.3.5
- A server that supports bot commands (for example .warstormbot)

No other addons required.

---

## Notes

- The addon manages class counts, not individual specs
- Spec selection is still done using the existing spec buttons
- Designed for private servers with bot support
